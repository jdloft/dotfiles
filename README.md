# jdloft: dotfiles
My Stuff

## Install
HTTPS
```
git clone https://github.com/jdloft/dotfiles.git ~/.dotfiles && ~/.dotfiles/bin/dotfiles
```
SSH
```
git clone git@github.com:jdloft/dotfiles.git ~/.dotfiles && ~/.dotfiles/bin/dotfiles
```
